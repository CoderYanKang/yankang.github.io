结构体 C中用户可自定义的数据类型，可以存放不同类型的值（数组用于存放相同的数据类型）。开发者可以根据实际情况创建一个新类型。

结构体可以包含结构体，在c++ 中结构体还可以包含函数，具有了类的部分功能。
